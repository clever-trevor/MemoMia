# MemoMia!
A simple note taking app with a folder structure and unlimited nesting.
* Customisable icons / colour for each level
* Support for tables, images, bullets, alignment, hyperlinks, emojis
* Notebook cached on local device so usable in case of no internet
* Import from Onenote capability

Built entirely in Google AI Studio

# Run and deploy your AI Studio app

This contains everything you need to run your app locally.

View your app in AI Studio: https://ai.studio/apps/edc00a38-45bc-47a8-94bd-a10cd673fd88

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`
